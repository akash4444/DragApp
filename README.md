## Installation

1. **Clone the repository:**

   ```sh
   git clone https://github.com/akash4444/DragApp.git
   ```

2. **Navigate to the project directory:**

   ```sh
   cd your-repo-name
   ```

3. **Install dependencies:**

   ```sh
   npm install
   ```

## Running the App

1. **Start the development server:**

   ```sh
   npm start
   ```

2. **Open your browser and visit:**

   ```
   http://localhost:3000
   ```
